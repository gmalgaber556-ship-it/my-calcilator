## الطريقة الثانية: بناء APK (للنشر على Google Play)

إذا كنت تريد ملف APK حقيقي:

### الخطوة 1: استخدم خدمة Expo EAS Build (مجاني)
```bash
# في مجلد المشروع
npm install -g eas-cli
eas login
eas build --platform android
```

### الخطوة 2: انتظر البناء
- سيأخذ 10-15 دقيقة
- ستحصل على رابط تحميل APK

### الخطوة 3: ثبت على هاتفك
- حمل الملف على هاتفك
- افتح الملف وثبته
